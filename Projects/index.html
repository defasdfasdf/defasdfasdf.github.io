Project Billfold

Windows

We think that it's important for Windows users to also have a QT wallet. This will probably (and hopefully) be finished way before Q1 even starts because of high demand. The QT wallet is the minimum wallet we'll be creating for Windows users

Linux

If you're using Linux you should not automatically have to use a QT wallet. We'll be creating a prebuild qt Linux wallet in or before Q1 of 2018. We'll also be creating a Electrum wallet for Linux as we understand that there are a lot of people who like to use Electrum over QT

Possibly more

In case of high demand or if we think it's neccessary we'll be looking into, if not creating, wallets for different platforms. We are already thinking about mac OS wallets but will have to see how things turn out.

Project Perdu

Anonimity

As you may know, SHIELD is at its core an anonymous coin. We care about our identity and we can only imagine how much our users care about theirs. This is why we have Project Perdu in place. Project Perdu is the project that will set SHIELD apart from all the other so-called anonymous coins in the field of anonimity. We'll be introducing a combination of already existing (and working) protocols. One of those will be introduced in the way of a masternode system (but we may have to wait untill that is ready as Boo is coming later.)
