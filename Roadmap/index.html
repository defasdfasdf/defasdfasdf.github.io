Project Billfold

Windows

We believe it's important for Windows users to also have access to a QT wallet. Due to high demand, this aspect of Project Billfold will likely be completed ahead of schedule. The QT wallet is planned to be the minimum wallet created for Windows users.

Linux

Linux users will not be confined to using a Windows QT wallet. We will be creating a prebuild QT Linux wallet during, or before Q1 of 2018. We also intend to create an Electrum wallet for Linux, as we understand that many Linux users prefer to use Electrum over QT.

Possibly more

In the case of high demand, or should we decide it necessary, we will be looking into, if not creating, wallets for various other platforms. A MacOS wallet is already being investigated, however its completion will be dependant on resource allocation for other aspects of SHIELD.

More information

Project Globalize

iOS

In order to most effectively allocate the time and resources of the development team, we do not currently plan to launch an iOS specific wallet application. Instead, we will be working with proven and trusted ‘multi-currency’ wallet applications already available for iOS to add SHIELD currency support.
When resources allow, we will create a dedicated SHIELD iOS application.

Android

Android will take precedent for mobile wallet development. In addition to the SHIELD Android wallet, we plan to work with multi-currency wallets available on the platform, to ensure our users can access their SHIELD currency in whichever way best suits them.

Project Sharding

Project Sharding will bring scalability improvement to SHIELD. With its origins in Ethereum, Sharding utilises validators to validate shards and subsequently decreases the size of the SHIELD blockchain an individual user needs to store.
While project Sharding represents on of the more technically difficult stages to implement, we beleive it will offer a significant improvement to SHIELD, and provide an excellent foundation to carry SHIELD into the future.
